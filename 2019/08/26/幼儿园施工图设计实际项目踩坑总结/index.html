幼儿园施工图设计实际项目踩坑总结

韩亦韶

< 我以前觉得，用钢琴弹出8位GBA游戏比如魂斗罗、超级玛丽和马戏团的背景音乐很酷。

保健观察室

  • 对于保健观察室最小使用面积的要求是和幼儿园的规模相关的，小型、中型和大型幼儿园中的保健观察室的最小使用面积的要求分别是12 m2、12 m2和15 m2。
  • 保健观察室设置应符合下列规定;
  1. 应设有一张幼儿床的空间；
  2. 应与幼儿生活用房有适当的距离并应与幼儿活动路线分开；
  3. 宜设单独出入口；
  4. 应设给排水设置；
  5. 应设独立的厕所，厕所内应设幼儿专用蹲位和洗手盆。

室内空调柜机、挂机和外机

  • 活动室和寝室合并之后的面积要求不小于120 m2，需要的空调柜机至少要两台，而且还是比较贵的，性能更好的那种。幼儿园、托儿所的相关规范有关空调室外机安装位置的要求：应安装在室外地面或通道地面2.0m以上，且幼儿无法接触的位置。我还没搞懂的是一个房间是必须要设柜机还是可以设挂机是有什么讲究还是可以随便设置。
